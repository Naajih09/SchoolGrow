# SchoolGrow

Multi-tenant school result and student management platform scaffold.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- Zustand
- TanStack Query
- React Hook Form
- Zod

## Getting started

1. Install dependencies with `pnpm install`.
2. Add `.env` values for `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
3. Run `pnpm dev`.
